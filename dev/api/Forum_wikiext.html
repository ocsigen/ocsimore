module Forum_wikiext : sig..end

Provides a function to register the wikicreole extensions for forums. Author(s): Vincent Balat, Boris Yakobowski

val register_wikiext : 
  Forum_widgets.message_widget * Forum_widgets.thread_widget *
  Forum_widgets.message_list_widget * Forum_widgets.threads_list_widget ->
  unit

This function registers the following wiki extensions:

  • <<forum_message>>
  • <<forum_thread>>
  • <<forum_message_list>>
  • <<forum_threads_list>>
